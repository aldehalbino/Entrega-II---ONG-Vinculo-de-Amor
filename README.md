# Entrega II — Vínculo de Amor (CSS3)

Projeto modernizado com Design System (CSS Custom Properties), Grid de 12 colunas, 5 breakpoints, navegação com dropdown e menu hambúrguer, componentes (cards, botões, formulários com validação visual, alerts/toast/modal) e estrutura modular de CSS.

## Estrutura
```
/css
  variables.css  # Design System
  base.css       # Reset + globais
  layout.css     # Grid 12 + breakpoints
  components.css # Nav, cards, botões, forms, feedback
  responsive.css # Ajustes específicos por seção
/js
  script.js      # Dropdown, hambúrguer, modal, toast
/imagens         # suas imagens (mantém nomes/paths)
/videos          # seus vídeos
```
## Como usar
Abra `index.html`. Todas as páginas compartilham cabeçalho/rodapé iguais.
